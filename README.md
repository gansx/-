# 3D Gesture Christmas Tree

这是一个带有照片的 3D 手势圣诞树项目。

## 在线访问 (GitHub Pages)

为了方便访问，主文件已重命名为 `index.html`。

### 如何发布

1.  **确保代码已推送**：代码已经提交到 GitHub 仓库。
2.  **开启 GitHub Pages**：
    *   打开 GitHub 仓库页面 -> **Settings** (设置) -> 左侧 **Pages**。
    *   **Source** 选择 `Deploy from a branch`。
    *   **Branch** 选择 `main` 分支，文件夹 `/ (root)`，点击 **Save**。
3.  **获取链接**：
    *   刷新页面，顶部会显示链接，例如：`https://<你的用户名>.github.io/<仓库名>/`。
    *   直接点击该链接即可访问。
